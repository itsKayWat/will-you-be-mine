=================

A playful web application for asking someone out on a date.

PURPOSE:
- Create a memorable moment when asking someone out
- Add fun and interactivity to digital dating
- Make the experience shareable and unique

INSTALLATION:
1. Download the files
2. Run: python requirements.py
3. Open index.html in a browser

REQUIREMENTS:
- Web browser
- Python 3.x (for setup)
- Node.js (optional, for notifications)
- Google Cloud account (optional)

USE CASE EXAMPLE:
John wants to ask Sarah out but wants to make it special. Instead of 
sending a regular text, he customizes this app and sends her the link. 
Sarah opens it to find a fun, interactive experience that makes the 
moment memorable, regardless of her answer.
